#  Pontua+ - Plataforma de Gamificação Educacional

Sistema inovador para combater a evasão escolar através de gamificação e recompensas.

##  Funcionalidades
- Sistema de pontos por desempenho acadêmico
- Rankings interativos (Bronze, Prata, Ouro, Diamond)
- Dashboard com métricas de desempenho
- Sistema de recompensas e conquistas

##  Tecnologias
- Java + Spring Boot
- MySQL
- Thymeleaf + Bootstrap

##  Sistema de Pontuação
- **Notas**: 35 pontos máximos
- **Frequência**: 15 pontos máximos  
- **Atividades Extras**: 50 pontos máximos
